# Tema do Projeto

Sistema de Vendas Online para Comércio Local utilizando Java RMI.

O sistema implementa um modelo cliente-servidor utilizando Remote Method Invocation (RMI), permitindo que clientes remotos realizem operações sobre produtos disponíveis em uma loja virtual.

---

# Objetivo

Implementar uma aplicação distribuída utilizando Java RMI seguindo o modelo request-response descrito no trabalho da disciplina.

A aplicação permite:

- Listagem de produtos;
- Busca de produtos;
- Compra de produtos;
- Cálculo de valor total da compra.

---

# Tecnologias Utilizadas

- Java
- Java RMI
- Programação Orientada a Objetos
- Serialização Java (`Serializable`)
